## 1. Storybook이란?

Storybook은 컴포넌트를 개발하고 문서화하는 도구입니다. `Storybook` 은 컴포넌트를
독립적으로 테스트하고 문서화하며, 컴포넌트를 개발할 때 빠르게 반복할 수 있도록 도와줍니다.
Storybook은 React, Vue, Angular, Svelte 등 다양한 프레임워크를 지원합니다. `기술부채`
contentlayer.config.ts 이번에는 Next.js에서 **_srotybook_**을 사용하는 방법을 알아보겠습니다.

### 1-1. Storybook 설치

Next js 설치 단계는 건너뛰고 Storybook 설치 단계부터 시작하겠습니다.

```bash
import { http, HttpResponse } from 'msw';

const BASE_URL = 'https://jsonplaceholder.typicode.com';

export const handlers = [
  http.get(`${BASE_URL}/users`, ({ cookies, request, params }) => {
    // Body, Query, Path Parameter를 사용하는 모의 서버를 구축하려면 콜백 함수의 매개변수를 사용하시면 됩니다.

    return HttpResponse.json(
      [
        {
          id: 1,
          name: '권용빈',
          age: 21,
        },
        {
          id: 2,
          name: '유재석',
          age: 40,
        },
      ],
      {
        status: 200,
      },
    );
  }),
];
```

현재 기준으로 Storybook 8버전이 설치됩니다.

만약 미리 설치된 7+버전 미만에서 사용한다면 다음과 같이 공식문서를 참고해주세요. [Storybook 7+버전 미만에서 업그레이드 방법 - 공식문서](https://storybook.js.org/docs/get-started/nextjs#in-a-project-with-storybook)
